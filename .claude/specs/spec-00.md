# Spec 00: Project overview

## Goal

Provide dynamic, detailed SketchyBar widgets for local CLI coding-agent sessions.

## Sequence

1. Spec 01: Claude Code + tmux event/state/rendering MVP.
2. Spec 02: Codex adapter and agent-specific configuration.
3. Spec 03: notifications, richer details, and release hardening.

## Constraints

Local only, no screen scraping, safe hook configuration, and click always opens details before jump-back.
